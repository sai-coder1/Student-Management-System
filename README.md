# Student Management System 📚

A console-based CRUD application built in C as part of my learning journey in systems programming.

## Features
- ➕ Add students with duplicate ID protection
- 📋 Display all students with grades
- 🔍 Search by ID or Name
- ✏️ Update student details
- 🗑️ Delete students
- 📊 Sort by marks (highest to lowest)
- 🎓 Auto grade calculation (A+, A, B, C, D, F)
- 💾 File persistence — data saves and loads automatically

## How to Run

### Compile
gcc main.c -o student

### Run
./student        # Linux/Mac
student.exe      # Windows

## Tech Used
- Language: C
- Concepts: Structs, File I/O, Arrays, Functions, Bubble Sort
